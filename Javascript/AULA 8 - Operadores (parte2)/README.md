<p>Operadores relacionais (resultado true ou false):<ul>
<li>></li>
<li><</li>
<li>>=</li>
<li><=</li>
<li>==</li>
<li>!=</li>
</ul></p>

<p>Identidade:<ul>
<li>5=='5' true, pois == nao testa o tipo em JS, logo para isso existe o === (e o !==, analogamente) </br>
5==='5' false</li>
</ul></p>

<ul>
<li>!</li>
<li>&&</li>
<li>||</li>
</ul></p>
